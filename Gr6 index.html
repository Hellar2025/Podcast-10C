Hva er doping

Doping er produkter som gir kunstig forbedring av presentasjonsevnen, og anabole steroider er en type doping. Folk bruker steroider for å bli større raskere, fordi at de ønsker kjappe resultater. Doping er forbudt i konkurranseidrett, inkludert anabole steroider. Disse stoffene etterlinger effekten av testosteron, som er hromonet som bidrar til at gutter utvikler større muskler i puberteten. Anabole steroider kan gi alvorlige bivirkninger, spesielt for ungdom. Langvarig bruk eller høye doser kan føre til at ungdom blir lavere enn normalt, fordi skjelettet modnes for raskt. Vanlige bivirkninger inkluderer aggressiv oppførsel, kviser og økt hårvekst. Doping kan også øke risikoen for hjerteproblemer, og den raske muskelveksten kan skade sener og muskler. Bruk av anabole steroider forstyrrer balansen av kjønnshormoner. Man kan oppleve lavere testosteronnivåer, som kan føre til feminine trekk, impotens og redusert sexlyst. For kvinner kan for mye testosteron gi maskuline trekk som dypere stemme og mer kroppshår. Begge kjønn kan risikere sterilitet, altså vansker med å få barn.
